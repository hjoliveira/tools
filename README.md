# Tools

A collection of single-page tools that run entirely in the browser.

**Live site:** https://hjoliveira.github.io/tools/

## Available Tools

| Tool | Description |
|------|-------------|
| [Goodreads to Obsidian Converter](https://hjoliveira.github.io/tools/goodreads-to-obsidian-base/) | Convert your Goodreads library CSV export into Obsidian-compatible markdown files with YAML frontmatter |

## Development

Each tool is a self-contained HTML file with inline CSS and JavaScript. See [AGENTS.md](AGENTS.md) for guidelines on creating new tools.
